# OneManager

OneManager adalah aplikasi fullstack untuk manajemen user, forum, dan pengaturan akun.

## 📁 Struktur Folder
onemanager/ 
├── backend/ # ASP.NET Core API + JWT Auth 
├── frontend/ # React + Tailwind + Vite


## 🚀 Cara Menjalankan

### Backend (ASP.NET)
### Frontend (React JS)

```bash
cd backend
dotnet run

cd frontend
npm install
npm run dev
```

## ✨ Fitur

Login dengan JWT

Role: Admin & Member

CRUD Admin & Member

Forum (Post + Gambar)

Reset Password + Edit Role (Admin only)

Settings: ubah profil
